# AllBase

*AllBase* - база всех моих проектов, документации и заметок.  
Главная задача *AllBase* – собрать и структурировать все работы в одной базе.

## Структура *AllBase*:
- **Год**
  - *Обучение*
    - Предметная область
      - Книги
      - Документация (лично написанная)
      - Заметки

  - *Проекты*
    - *В Разработке*
      - Предметная область
    - *Pet-Проекты*
      - Предметная область
- **Результаты** (года и общий)
